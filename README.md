# CV-Builder

A modern, responsive Next.js application that allows users to easily build, customize, and export professional CVs/Resumes.

## Features

- **Dynamic Resume Building:** Add, edit, and organize your work experience, education, and skills.
- **Real-time Preview:** See your resume update instantly as you type.
- **Export to PDF:** Download your finished resume as a polished, print-ready PDF.
- **Responsive Design:** Works seamlessly on desktop and mobile devices.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS / CSS Modules
- **Font:** Geist (Optimized via `next/font`)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
